## Решение в circle.py
### 1-ая функция
В первой строчке импортирую библиотеку math для использования math.pi
### 2-ая функция
Функция area принимает значение r и с помощью его вычисляет площадь окружности по формуле math.pi*r*r
### 3-я функция
Функция perimeter также принимает r что также является радиусом окружности и с его помощью вычисляет периметр окружности по формуле 2*math.pi*r
## Решение в rectangle.py
### 1-ая функция
Функция Area принимает на вход два значения a, b которые являются сторонами прямоугольника и с помощью этих значений вычисляет площадь по формуле a*b пример: a=5, b=6, area = 5*6
### 2-ая функция
Функция perimeter принимает на вход также 2 значения a и b которые также являются сторонами прямоугольника и с помощью этих значений функция находит периметр прямоугольника по формуле (a + b)*2  например: a = 2, b = 3, perimeter = 10.
## Решение в square.py
### 1-ая функция
Функция Area принимает на вход одну переменную a которая является стороной квадрата и с помощью этой переменной считает площадь квадрата по формуле area = a^2, пример: a = 4, area = 4^2
### 2-ая функция
Функция perimeter принимает на вход также одно значение и с помощью него считает периметр квадрата по формле P=a*4 пример: a=4 P = 16.
